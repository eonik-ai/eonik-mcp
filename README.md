## What is eonik?

[eonik](https://www.eonik.ai) helps performance teams ship **finished, on-brand ads**: AI-native editor, agentic creative-testing planning, human approval on every cut. You direct; eonik does the busywork.

**Not** a budget autopilot, kill/scale engine, or performance oracle.

## The eonik MCP Server

This [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) server connects Claude Desktop and OpenClaw agents to your eonik workspace — research, creative readouts, and on-brand ad assembly workflows. **You approve every cut and every spend decision.**

### Capabilities

- **Research:** `search_ad_library`, `get_brand_context`, `get_insights_feed`
- **Readout:** `run_budget_audit`, `get_creative_autopsy` (signals and testing considerations — not autonomous spend actions)
- **Produce:** `create_ad_creation_run`, `compile_seed_spec`

Heavy lifting runs in eonik cloud. This server is an authenticated bridge only.
